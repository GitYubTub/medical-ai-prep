## Introduction to Unix

### Unix
Unix - a version of operating system
- can handle multiple users at the same time
- also supports multiple tasking
- a set of programs that links the computer and the user

Operating system / Kernel - a computer program that assigns the system resources and coordinates all internal details of the computer
- Users communicate with the kernel using a software program called the shell

The shell - acts as a translator between the user and the kernel, translating the user's input into a language that the kernel can understand

There are many variants of Unix: Ubuntu, Solaris Unix, HP Unix

Kernal: 
- Is the heart of the operating system
- Interacts with hardware and handles tasks input by the user

Shell:
- a utility that processes the user's requests
- Acts between the communication with the user and the Kernel
- When a user types a command into the terminal, the shell interprets the command and calls a program to complete the user's command
- The shell uses standard syntax for all commands
- Shells: C Shell, Bourne Shell, and Korn Shell

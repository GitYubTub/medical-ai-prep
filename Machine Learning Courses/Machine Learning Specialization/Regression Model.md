## Regression Model
- Regression models predict continuous numerical outputs (e.g., price = $220,000)
- Infinitely many possible outputs

### Notation and Training Set
- Input features are denoted as lowercase x (e.g., house size)
- Output targets are denoted as lowercase y (e.g., house price)
- The training set consists of m examples, each with an (x, y) pair
- Superscript notation (x^(i), y^(i)) refers to the "ith" training example.

### Supervised Learning Basics
- The learning algorithm produces a function, called the model (f), that takes new inputs (x) and outputs predictions (ŷ), estimates of the true output (y)

### Linear regression model
- fits a straight line to data points to predict continuous numerical values, such as house prices
- The function f can be represented as a linear function: f_w,b(x) = wx + b, where w and b are parameters that determine the prediction.

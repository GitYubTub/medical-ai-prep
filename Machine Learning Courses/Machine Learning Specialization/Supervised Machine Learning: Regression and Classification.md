## Supervised vs. Unsupervised Machine Learning

### What is Machine Learning
- Machine learning is defined as the field that enables computers to learn without explicit programming
  
### Two types of Machine Learning
- Supervised learning
- Unsupervised learning
- 
### Supervised learning
- involves algorithms that learn mappings from input (x) to output (y) by training on examples with known correct answers (labels)
- The goal is to systematically choose the best model to predict prices accurately
- Types of Supervised Learning Problems
  - **Regression**: a supervised learning technique used to predict continuous, numerical output values by modeling the relationship between input features and a target variable
    - Ex: Given data of house sizes and prices, the algorithm learns to predict the price of a new house based on its size
  - **Classification**: predict discrete categories or classes from a limited set of possible outputs
    - outputs can be non-numeric categories or numeric labels representing categories
    - Ex: The task is to classify tumors as benign (0) or malignant (1) based on features such as tumor size

### Unsupervised learning
- involves analyzing data without associated output labels
- The goal is to discover structure, patterns, or interesting groupings in the data without input examples
- An example of unsupervised learning:
  - Clustering algorithms group unlabeled data into clusters based on similarities found by the algorithm itself
      - Ex:
        - Google News grouping related articles by common keywords
        - clustering genetic data to identify types of individuals based on gene expression
  - Anomaly Detection: Identifies unusual events, useful for applications
  - Dimensionality Reduction: Compresses large datasets into smaller ones while preserving as much information as possible

## Command-Line Essentials

### Commands and Utilities:
- entered into the shells with particular arguments of what the user wants to do.
- input Commands and Utilities -> Shells -> Kernel (Unix) output
- Every command has a program written in the system, often provided by external sources (third parties)
- All commands have different output options

Files and Directories:
- All files in Unix are organized into files, and all files are organized into directories.
- The directories are organized into a tree structure called a file system

Unix System Startup:
- instantaneous
- prompts login immediately after startup
- after login, asks for a dollar prompt
